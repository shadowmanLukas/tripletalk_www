# TripleTalk WWW

Minimal Astro landing page for TripleTalk.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Cloudflare Pages

- Framework preset: Astro
- Build command: `npm run build`
- Output directory: `dist`
- Production branch: `main`
- Custom domain: `tripletalk.app`
